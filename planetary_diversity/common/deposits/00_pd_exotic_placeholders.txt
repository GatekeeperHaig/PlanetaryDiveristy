# These stub deposits exist so that PD main's vanilla deposit overrides can
# safely reference d_pd_ammonia_* (and future cryo deposits) in their
# terraforming_swap_types lists without producing "Invalid blocker swap type"
# log errors when the exotics submod isn't loaded.
#
# When the exotics submod IS loaded, its real definitions in
# planetary_diversity_exotics/common/deposits/15_pd_exotic_ammonia_deposits.txt
# override these stubs (later mod wins on duplicate identifiers).
#
# Pattern mirrored from planetary_diversity_ascension_worlds compatibility
# (see pd_ascension_deposits.txt): potential never satisfies and drop weight is
# zero, so the placeholder can never spawn or apply on its own.
